Large Language Models

Definition

Large Language Models (LLMs) sind fortschrittliche KI-Systeme, die mithilfe von maschinellem Lernen große Mengen an Textdaten analysieren, um Sprache zu verstehen und zu generieren. Sie sind darauf trainiert, Muster in Texten zu erkennen und können vielfältige Aufgaben wie Textvervollständigung, Übersetzung und Beantwortung von Fragen durchführen. Sie sind in der Lage, menschenähnliche Texte zu produzieren, indem sie kontextbezogene Informationen nutzen. Beispielsweise GPT-3 (Generative Pre-trained Transformer 3) ist ein LLM, der von OpenAI entwickelt wurde.

Funktionsweise

Um LLMs in eigene Webseiten einzubinden, habe ich die API von OpenAI genutzt. Die API ermöglicht es, sie so zu trainieren, dass nur bestimmte Texte generiert werden. So kann man die Texte auf bestimmte Themen oder Stile anpassen, so dass die Texte auf die Bedürfnisse der eigenen Webseite passen und nicht zu allgemein sind.

Die Idee

Wer kennt es nicht, es gibt ein besonderes Ereignis und man möchte einem Bekannten oder Verwandten eine Postkarte schicken. Doch beim schrieben fallen einem einfach nicht die passenden Worte ein oder es steht jedes Jahr das gleiche auf der Weihnachtskarte. Um dieses Problem zu lösen, wollte ich einen Postkarten-Schreiber erstellen, diese Webseite kann die durch eintragen der Namen und des Events automatisch einen personalisierten Postkarten-Texte generiert. So haben die Nutzer immer die passenden Worte und schreiben vielleicht wieder öffter eine Postkarte.

Vorgehensweise

Ich habe die Webseite erstellt, auf der die Nutzer die Namen und das Event eingeben können. Die Webseite ist schlicht und einfach gehalten, und mit der Schreibmaschinenschrift etwas oltschool. Ich habe das LLM dann so trainiert dass er immer relativ kurze Texte generiert, die sich auf die Namen und das Event beziehen. Bei jeden Mal wenn erneut auf senden geklickt wird sollte zudem ein neuer Text generiert werden. Ausserdem sollten die Texte immer positiv sein und passende Emojicons enthalten.

Schwierigkeiten

Die größte Schwierigkeit war es, wirklich nur Postkartentexte zu erhalten, so dass er beispielweise nicht auf eine Frage antwortet die in eine Spalte eingegeben wurde. Genrell musste ich viel experimentieren, um die gewünschten Ergebnisse zu erzielen.

Präsentation/Form/darstellung

Ich wollte den Postkarten Schreiber auf einer schlichten und einfach gehalten Webseite, damit die Nutzer sich auf den Text konzentrieren können. Mit der Schreibmaschinenschrift sollte sie wieder etwas oltschool wirken.

Mögliche anwendungen

Aber nicht nur zum generieren von Postkartentexten können LLMs genutzt werden sie könne in fast jedem Bereich der Textgenerierung eingesetzt werden. Beispielsweise könnten sie in der Kundenkommunikation eingesetzt werden, um automatisch auf Anfragen zu antworten oder zum umschreiben von Texten.

Fazit

Die Arbeit mit LLMs hat mir gezeigt, wie mächtig KI-Systeme sein können und wie vielfältig sie eingesetzt werden können. Es ist faszinierend, wie sie menschenähnliche Texte generieren können und wie sie in der Lage sind, komplexe Aufgaben zu bewältigen. Allerdings ist es auch wichtig sicherzustellen, dass sie verantwortungsbewusst eingesetzt werden. Eine Postkarte die mit KI geschrieben ist, kann lustig sein aber natürlich nicht die persönliche Note ersetzen. Durch dieses Projekt habe ich gelernt, LLMs in die eigene Webseite einzubauen und sie für spezifische Aufgaben zu trainieren.
